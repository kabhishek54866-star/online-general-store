## 📌 What Is This?

**Online General Store** is a production-grade, full-stack e-commerce platform built entirely from scratch. It ships with two separate frontends — a **Customer Storefront** for buyers and an **Admin POS Dashboard** for store owners — backed by a **Spring Boot REST API** and a **MySQL** database.

No shortcuts. No component libraries. Pure Java + React + Vanilla CSS.
